NodeGet 静态桶说明

什么是静态桶？

静态桶（Static Bucket）是 NodeGet 提供的静态文件托管功能。 你可以将 HTML、CSS、JavaScript、图片等任意静态文件上传到桶中， NodeGet 会通过 HTTP 提供服务。

使用方式

  1. 在「静态资源管理」中创建一个 Bucket
  2. 上传本地目录（支持 ZIP 包或文件夹选择）
  3. 开启 root 路由 后，该桶会响应所有根路径 GET 请求
  4. 通过 NodeGet 服务地址直接访问页面

支持的功能

  • ✓ 任意静态文件（HTML / CSS / JS / 图片）
  • ✓ 文件树浏览与在线编辑
  • ✓ ZIP 打包下载
  • ✓ 一键上传整个目录
  • ✓ root 路由（托管整站）
